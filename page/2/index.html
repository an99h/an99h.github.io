小米路由器3安装 Aria2 下载百度网盘

小米路由器3安装 Aria2 下载百度网盘

  1. chrmoe 插件 BaiduExporter，YAAW for Chrome/Aria2 Web UI
  2. 小米路由器刷开发板固件
  3. MIWiFi SSH下载 ssh.bin。开机捅菊花升级
  4. ssh root@192.168.31.1
  5. opkg安装 aria2
    1. 安装 opkg (scp ./opkg root@192.168.31.1:/data)
    2. opkg install https://downloads.openwrt.org/chaos_calmer/ 15.05.1/ar71xx/generic/packages/base/libc_0.9.33.2-1_ar71xx.ipk（aria2 依赖）
    3. opkg install aria2 --force-depends

  6. 配置 aria2

    1. 创建/etc/aria2/aria2.conf

2. 开机自启:/etc/rc.local exit(0)前加上`aria2c --conf-path=/etc/aria2/aria2.conf -D`
3. 手动启动 aria2：`aria2c --conf-path=/etc/aria2/aria2.conf`
  1. 安装 chrome BaiduExporter插件后百度云盘会出现 aria2下载按钮
  2. 配置BaiduExporter，YAAW for Chrome/Aria2 Web UI
    JSON-RPC Path：http://192.168.31.1:6800/jsonrpc
  3. YAAW for Chrome/Aria2 Web UI插件，显示下载进度

git 操作总结

命令行git总结

安装git

homebrew安装git，homebrew具体操作教程

安装完成后，还需要最后一步设置

创建本地版本库

创建本地仓库文件夹

初始化git仓库

创建好后是一个空的仓库，$ ls -la可以发现多了.git隐藏目录

创建远程仓库

  • 登陆GitHub，创建于本地仓库同名的仓库
  • 创建好后找到`git@github.com:yourename/testGit.git`记住。

提交本地仓库到远程仓库

创建SSH Key

可以在用户主目录里找到.ssh目录，里面有id_rsa和id_rsa.pub两个文件，这两个就是SSH Key 的秘钥对，id_rsa 是私钥，不能泄露出去，id_rsa.pub是公钥，可以放心地告诉任何人。

设置SSH Key

登陆GitHub，打开“Account settings”，“SSH Keys”页面
然后，点“Add SSH Key”，在Key文本框里粘贴id_rsa.pub文件的内容

本地仓库与远程仓库关联

首次将本地仓库提交到远程仓库

git基本命令

  • 从GitHub上克隆项目到本地: git clone git@github.com:yourename/testGit.git
  • 与GitHub远程仓库同步: git pull
  • 查看文件变更: git status
  • 提交代码到本地缓存: git commit -m 'description'
  • 提交代码到远程GitHub仓库:git push
  • 查看标签: git tag
  • 添加标签: 轻量级标签git tag tagName, 带注释标签git tag -a tagName -m 'description'
  • 删除标签: git tag -d tagName
  • 提交标签到GitHub中: git push origin --tags
  • 列出分支: git branch
  • 切换分支: git checkout master
  • 提交分支: git push origin branchName
  • 删除分支: git branch -d branchName,强制删除分支 git branch -D branchName
  • 合并分支: git merge branchName

在GitHub Pages上搭建Hexo博客

前言

作为前端小白只能在前人基础上搭建网站创建自己博客。为什么会选择在GitHub Pages上搭建博客？因为空间免费，操作简单，用来做搭建一个博客再好不过了。在GitHub Pages创建博客可以认为是用户编写，托管在GitHub上的静态网页。
Hexo是一个简单、快速、强大的博客发布工具，支持Markdown格式。

前期准备

注册GitHub账号

http://www.github.com/ 注册过程比较简单
在GitHub上新建xxx.github.io仓库（xxx为自己用户名）

安装git

安装hexo

购买域名

这一步可选，去阿里云买个不常见的顶级域名很便宜的，我的.space域名去年买的，首年4元，又续了两年,也不算贵总共26元。

本地创建hexo后台

1.创建hexo博客文件夹

Hexo会自动在目标文件夹建立网站所需要的所有文件

2.编译当前博客

3.启动本地server，浏览器输入localhost:4000

如果进入博客首页，可以进入下一项设置。

配置hexo _config.yml

1.最后几行配置如下，==注意：后面要有空格==

2.首先配置本地git账户，下次再介绍。

3.推送本地静态博客到GitHub Pages。

4.在浏览器输入xxx.github.io,不出问题应该可以打开和之前本地一样的界面

绑定自己域名

  1. 在阿里云域名解析，添加CNAME记录,记录值为xxx.github.io.
  2. 在本地hexo/source/目录下新建CNAME文件没有后缀名，内容为自己注册的域名。
  3. 再次编译自己博客

5.推送完成用自己域名试试，应该大功告成了。
