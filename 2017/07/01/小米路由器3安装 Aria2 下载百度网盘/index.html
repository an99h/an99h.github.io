教程&总结

小米路由器3安装 Aria2 下载百度网盘

  1. chrmoe 插件 BaiduExporter，YAAW for Chrome/Aria2 Web UI
  2. 小米路由器刷开发板固件
  3. MIWiFi SSH下载 ssh.bin。开机捅菊花升级
  4. ssh root@192.168.31.1
  5. opkg安装 aria2
    1. 安装 opkg (scp ./opkg root@192.168.31.1:/data)
    2. opkg install https://downloads.openwrt.org/chaos_calmer/ 15.05.1/ar71xx/generic/packages/base/libc_0.9.33.2-1_ar71xx.ipk（aria2 依赖）
    3. opkg install aria2 --force-depends

  6. 配置 aria2

    1. 创建/etc/aria2/aria2.conf

2. 开机自启:/etc/rc.local exit(0)前加上`aria2c --conf-path=/etc/aria2/aria2.conf -D`
3. 手动启动 aria2：`aria2c --conf-path=/etc/aria2/aria2.conf`
  1. 安装 chrome BaiduExporter插件后百度云盘会出现 aria2下载按钮
  2. 配置BaiduExporter，YAAW for Chrome/Aria2 Web UI
    JSON-RPC Path：http://192.168.31.1:6800/jsonrpc
  3. YAAW for Chrome/Aria2 Web UI插件，显示下载进度
